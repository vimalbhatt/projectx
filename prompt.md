# Goal #
Web App for secret chat

# Requirements #
- create a streamlist based chat app which allows two users two users to chat with each other.
- Users are 'A' and 'B'. 
- Either you could login using A or B. Each would have a different password.
- The chat would be persistent
- Both users have option to send message, read other's message, clear chat.
- The message would be entered normally by sender i.e. non-encrypted.
- The message would get encrypted when sent and appear encrypted
- The reader has to select the message text and press certain key combination to decrypt it. It would return to encrypted state when user deselect the message.
- The key combination would be unique to both user and user can change his/her key combination and login password.
- Once decrypted, the message becomes READ from UNREAD
- The message would be transferred, stored and retrived in encrypted mode only. It would only get decrypted on the page while user is reading it by pressing certain key combination. 
- The decrypted message would get encrypted again after 10 seconds.
